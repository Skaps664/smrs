## Glassmorphic Effect Details

```
HERO STAT CARDS:

Background: white/10 (10% opacity white)
Backdrop Filter: blur(16px)
Border: None or 1px white/20
Border Radius: xl (12px)
Padding: 16px
Shadow: None (relies on backdrop)

Example CSS:
.stat-card {
  background: rgba(255, 255, 255, 0.1);
  backdrop-filter: blur(16px);
  border-radius: 12px;
  padding: 16px;
}

VIDEO BUTTON:

Background: white/20
Backdrop Filter: blur(16px)
Hover: white/30
Border Radius: full (9999px)
Padding: 12px 24px
Transition: all 0.3s

Example CSS:
.video-button {
  background: rgba(255, 255, 255, 0.2);
  backdrop-filter: blur(16px);
  border-radius: 9999px;
  transition: all 0.3s;
}
.video-button:hover {
  background: rgba(255, 255, 255, 0.3);
}
```
