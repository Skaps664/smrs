## Glassmorphic Effect Details

```
HERO STAT CARDS:

Background: white/10 (10% opacity white)
Backdrop Filter: blur(16px)
Border: None or 1px white/20
Border Radius: xl (12px)
Padding: 16px
Shadow: None (relies on backdrop)

Example CSS:
.stat-card {
  background: rgba(255, 255, 255, 0.1);
  backdrop-filter: blur(16px);
  border-radius: 12px;
  padding: 16px;
}

VIDEO BUTTON:

Background: white/20
Backdrop Filter: blur(16px)
Hover: white/30
Border Radius: full (9999px)
Padding: 12px 24px
Transition: all 0.3s

Example CSS:
.video-button {
  background: rgba(255, 255, 255, 0.2);
  backdrop-filter: blur(16px);
  border-radius: 9999px;
  transition: all 0.3s;
}
.video-button:hover {
  background: rgba(255, 255, 255, 0.3);
}
```

## Video Embed Aspect Ratio

```
RESPONSIVE 16:9 CONTAINER:

<div style="position: relative; padding-bottom: 56.25%">
  <iframe 
    style="position: absolute; top: 0; left: 0; width: 100%; height: 100%"
    src="https://www.youtube.com/embed/VIDEO_ID"
  />
</div>

Calculation: 9/16 = 0.5625 = 56.25%

This maintains perfect 16:9 aspect ratio at any screen size!
```
